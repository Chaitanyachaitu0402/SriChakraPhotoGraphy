# Sri Chakra Photography — Wedding Photography Website

This project is a fully designed wedding photography website created for Sri Chakra Photography.
The website showcases various photography services such as Weddings, Pre & Post Weddings, Maternity, Newborn, Events, and more.
It highlights visual storytelling, elegant layouts, and a modern, dark-themed UI that reflects the brand style.

Technologies Used

HTML – Structure & layout of the website

CSS – Styling, animations, typography & responsiveness

JavaScript – Interaction, dynamic elements & functional UI behaviors

Features

Clean & elegant photography portfolio sections

About section with photographer introduction

Professional service listings

Responsive design suitable for desktop & mobile

Crafted typography and visual experience focused on storytelling

Purpose

This website is built to represent the photography brand, showcase work, and provide a visually appealing experience for visitors planning their memorable events.
